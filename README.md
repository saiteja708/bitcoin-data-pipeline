# Bitcoin Data Pipeline Project

## 📌 Overview
This project builds a complete ETL pipeline to collect, process, and visualize Bitcoin price data.

## 🎯 Business Case
Bitcoin prices are highly volatile. This project analyzes price trends to support data-driven decisions.

## ⚙️ Technologies Used
- Python (Pandas, Requests, Matplotlib)
- Power BI
- PostgreSQL (optional)

## 🔄 Pipeline Flow
API → Data Cleaning → CSV → Visualization → Dashboard

## 📊 Outputs
- bitcoin.csv
- 5 charts (stored in visuals folder)
- Power BI dashboard
